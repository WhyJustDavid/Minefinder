# Minefinder

Minefinder is a clone of Minesweeper I first made back in 2024 in a kid's coding camp.
Two years later, in 2026, I'm in my teens, and I found the flash drive with my files and decided to try to finish what I started.

To edit the original files, I had to download the GameMaker 8.1 Installer from here:
https://archive.org/details/GameMaker81
